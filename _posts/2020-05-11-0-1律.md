## 0-1 律

PTE5的 Page 81

定义尾$\sigma$-域：$\mathcal{T} = \bigcap_n\mathcal{F}_n' = \bigcap_n \sigma(X_n,X_{n+1},\dots)$. 

- **Kolmogorov 0-1 律**：如果$X_1,X_2,\dots$是独立的随机变量，$A\in\mathcal{T}$, 那么
  $$P(A) = 0 \ \text{or} \ 1$$
  - 证明要点：$A$与自己独立，也就是$P(A\cap A) = P(A)P(A)$
    - 如果$A\in\sigma(X_1,\dots,,X_k)$, $B\in\sigma(X_{k+1},X_{k+2},\dots)$, 那么$A$与$B$独立。
    - $A\in\sigma(X_1,X_2,\dots)$, $B\in T$那么$A$与$B$独立。

定义可置换$\sigma$-代数：如果通过有限次置换不影响的事件。
- **Hewitt-Savage 0-1 律**：如果$X_1,X_2,\dots$是独立同分布的随机变量，$A\in\mathcal{E}$, 那么$P(A)\in \{0,1\}$. 

## 大偏差

PTE5 P105

记$X_i$为 i.i.d. 序列，$S_n$是部分和。我们要考察下面概率的收敛速率 $P(S_n>n\alpha)\to 0$, 其中$a$大于均值$\mu$. 最终要证明的是，如果矩生成函数$\phi(\theta)$对于某个$\theta$有$\phi(\theta)<\infty$, 那么收敛速度就是指数级的。并且可以找到一个特征指数：
$$\gamma(a) = \lim_{n\to \infty}\frac{1}{n}\log P(S_n \geq na).$$ 

- 记$\pi_n = P(S_n\geq na)$, 那么由独立性：
  $$\pi_{m+n}\geq P(S_m\geq ma, S_{n+m}-S_m\geq na) = \pi_m\pi_n$$
- 记$\gamma_n = \log \pi_n$, 可以让乘法变成加法。

下面有两个引理：
1. 如果$\gamma_{m+n} \geq \gamma_{m}+\gamma_{n} \text { then as } n \rightarrow \infty, \gamma_{n} / n \rightarrow \sup _{m} \gamma_{m} / m$. 
2. 记$\kappa(\theta)=\log\phi(\theta)$. 如果$a>\mu$, $\theta>0$很小，那么$a \theta-\kappa(\theta)>0$.

### 大偏差的例子

1. 正态分布
   $$\begin{aligned}
       Ee^{\theta x} &= \int e^{\theta x}(2 \pi)^{-1 / 2} \exp \left(-x^{2} / 2\right) d x \\&=\exp \left(\theta^{2} / 2\right) \int(2 \pi)^{-1 / 2} \exp \left(-(x-\theta)^{2} / 2\right) d x
       \\&= \exp(\theta^2/2)
   \end{aligned}$$
2. 参数的 $\lambda$ 指数分布
   $$\int_{0}^{\infty} e^{\theta x} \lambda e^{-\lambda x} d x=\lambda /(\lambda-\theta)$$
   
   $$\varphi^{\prime}(\theta) \varphi(\theta)=1 /(\lambda-\theta)$$ 
   
   and

    $$F_{\theta}(x)=\frac{\lambda}{\lambda-\theta} \int_{0}^{x} e^{\theta y} \lambda e^{-\lambda y} d y$$
